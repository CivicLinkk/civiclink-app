# CivicLink App

Empowering local democracy through civic engagement.

## Features

- 🔧 Community issue reporting
- 👮 Police interaction documentation
- 🛂 ICE activity reporting (enhanced with GPS, video, frequency tracking)
- 📅 Civic events calendar
- 📊 Personal dashboard

## Setup

1. Clone this repository
2. Install dependencies: `npm install`
3. Create `.env.local` file with Supabase credentials
4. Run development server: `npm run dev`

## Environment Variables
```
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```

## Deployment

Deploy to Vercel with one click or push to GitHub and connect to Vercel.
